## 🎯 Описание проекта

### Проект №1: Скрапер
Собирает данные о книгах с сайта `books.toscrape.com`:
- Название книги
- Цена (в фунтах)
- Рейтинг (от 1 до 5 звёзд)
- Наличие на складе

Результаты сохраняются в:
- `CSV` — структурированные данные
- `TXT` — аналитический отчёт
- `PNG` — график распределения цен

### Проект №2: API
Предоставляет REST API для доступа к собранным данным:

| Возможность | Описание |
|-------------|----------|
| **Фильтрация** | по цене, рейтингу, наличию |
| **Сортировка** | по цене, рейтингу, названию |
| **Пагинация** | ограничение выдачи (макс 50 записей) |
| **Поиск** | по названию книги |
| **График** | получение PNG изображения |
